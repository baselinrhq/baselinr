# Baselinr v0.8.0 Release Notes

## New Features

* **Quality Studio**: Complete rebrand of dashboard to "Quality Studio" with emphasis on no-code data quality setup and configuration management
* **Configuration Management Hub**: Centralized UI for managing all Baselinr configuration through visual forms
* **Connection Wizard**: Interactive wizard for setting up and testing database connections
* **Storage Configuration UI**: Visual interface for configuring Baselinr storage settings
* **Table Selection & Discovery UI**: Visual interface for selecting tables and discovering tables using patterns
* **Profiling Configuration UI**: Configure profiling settings, metrics, and table-level overrides through visual forms
* **Validation Rules Builder**: Visual interface for creating and managing data quality validation rules
* **Validation Dashboard**: Comprehensive dashboard for viewing validation results, rules, and failure analysis
* **Drift Detection Configuration UI**: Visual interface for configuring drift detection strategies, baselines, and thresholds
* **Anomaly Detection Configuration UI**: Configure anomaly detection methods, learning settings, and expectation thresholds
* **Alert Hooks Configuration UI**: Manage alert hooks for notifications and event handling
* **Config Editor with YAML Preview**: Split-view editor with visual forms and real-time YAML preview
* **Configuration Version History**: Track, compare, and restore previous configuration versions with diff visualization
* **Root Cause Analysis Dashboard**: Interactive UI for analyzing data quality issues with temporal correlation and lineage analysis
* **Smart Recommendations UI**: Visual interface for AI-powered table and column recommendations
* **Enhanced Data Lineage View**: Improved lineage visualization with better navigation and filtering
* **Lineage-Aware Prioritization**: Intelligent prioritization of tables based on lineage relationships and downstream impact

## Improvements

* **Enhanced Dashboard Views**: Improved UI for runs, drift, tables, and home dashboard with better filtering and visualization
* **Validation Rules API**: New API endpoints for managing validation rules programmatically
* **Configuration State Management**: Centralized state management for configuration across the Quality Studio
* **UI Component Library**: Reusable component library for consistent design across Quality Studio
* **Dark Theme**: Complete dark theme implementation with glass morphism effects and improved readability
* **Breadcrumb Navigation**: Added breadcrumb navigation across configuration pages for better user orientation
* **Test Coverage**: Enhanced test coverage for all new UI components and configuration features
* **CI/CD Integration**: Added frontend testing and linting to GitHub Actions workflow

## Maintenance

* **Code Quality**: Linting and formatting improvements across frontend and backend
* **Documentation**: Updated all documentation to reflect Quality Studio branding and new features
* **Bug Fixes**: Fixed test failures and linting errors in configuration components
* **Performance**: Optimized configuration loading and state management
