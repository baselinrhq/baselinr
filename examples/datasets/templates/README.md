# Dataset Templates

Starter templates for creating dataset configuration files. Copy and customize.

## Table Template (`table_template.yml`)
Use for a single table with optional profiling, drift, validation, anomaly, and column overrides.

## Schema Template (`schema_template.yml`)
Use for defaults that apply to every table in a schema.

## Database Template (`database_template.yml`)
Use for broad defaults across a database (lowest precedence before global defaults).
